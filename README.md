# Habit & Streak Tracker

A simple frontend Habit Tracker web application built using HTML, CSS, and JavaScript.

## Features
- Add habits
- Track streaks
- Simple and clean UI
- Multiple pages navigation

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Pages
- Home Page
- Add Habit Page
- Habits Page
- Streak Page
